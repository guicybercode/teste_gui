# Placeholder Images Directory

This directory is intended for your custom images. The gallery in index.php will display images from this directory.

If images are not found, the application will automatically use placeholder images from an online service.

To add your own images:
1. Place image files (jpg, png, etc.) in this directory
2. Name them: image1.jpg, image2.jpg, image3.jpg, image4.jpg
3. Or modify index.php to use your custom image names

Supported formats: JPG, PNG, GIF, WebP
